Component: AnimatedCounter

Language: TypeScript

{

================================================================================
ANIMATION DETAILS
================================================================================

Easing Function: easeOutQuad
  - Formula: 1 - (1 - t)²
  - Effect: Starts fast, decelerates smoothly
  - Provides natural, polished feel

Animation Timing:
  - Uses requestAnimationFrame (targets 60fps)
  - Frame-based calculation using elapsed time
  - Automatic cleanup on unmount

Number Formatting:
  - Uses Number.toLocaleString('en-US')
  - Adds comma separators every 3 digits
  - Examples: 1000 → "1,000", 38400 → "38,400"

Visual Integration:
  - Opacity fade-in: 0 → 1 over 0.5 seconds
  - Inline-block display for proper spacing
  - Font weight: semibold (default)

================================================================================
TROUBLESHOOTING QUICK REFERENCE
================================================================================

Issue: Animation doesn't start
Solution: Check component is mounted, verify 'end' is a number

Issue: Numbers not formatting with commas
Solution: Verify browser supports toLocaleString() (all modern browsers do)

Issue: Animation timing feels off
Solution: Adjust 'duration' prop (typical: 1500-3000ms)

Issue: Performance problems with many counters
Solution: Use VisibleAnimatedCounter to animate only visible counters

Issue: Number looks wrong
Solution: Check 'end' prop is correct, verify prefix/suffix are strings
