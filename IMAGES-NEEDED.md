## Color Guidelines for All Images

- **Primary Accent**: `#14B8A6` (Teal-500)
- **Secondary Accent**: `#0D9488` (Teal-600)
- **Highlight**: `#0EA5E9` (Sky-500)
- **Neutral Light**: `#F8FAFC` (Slate-50)
- **Neutral Dark**: `#1E293B` (Slate-800)

All images should feel:
- Professional and trustworthy
- Modern and clean
- Active and energetic
- Approachable for B2B audience

---

## Notes

1. All lifestyle photography should feature diverse representation
2. Avoid stock photo "stiffness" - aim for authentic moments
3. Images should work on both light and dark backgrounds where applicable
4. Consider WebP format for web delivery (with PNG/JPG fallbacks)
5. Optimize all images for web (compress without quality loss)
