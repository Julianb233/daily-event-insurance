# Plan 03-01: Lead Detail Page with Communication Timeline

**Phase:** 3 - Lead Management Enhancement
**Status:** PLANNED
**Estimated Complexity:** High

## Objective

Build comprehensive lead detail page showing lead information, communication history timeline, and action buttons for calling/SMS.

## Prerequisites

- Phase 1 complete (database verified)
- Phase 2 complete (scripts UI pattern)
- API `/api/admin/leads/[id]` functional

## Implementation

### File to Create

`app/(admin)/admin/leads/[id]/page.tsx`

### Features

1. **Lead Information Card**
   - Name, email, phone, business info
   - Status badge with dropdown to change
   - Interest level indicator
   - Lead value ($40 initial → converted value)

2. **Communication Timeline**
   - Chronological list of all interactions
   - Icons for call/SMS/email
   - Call duration, disposition, outcome
   - SMS content preview
   - Expandable details

3. **Action Buttons**
   - "Call Now" button → triggers LiveKit call
   - "Send SMS" button → opens SMS modal
   - "Schedule Follow-up" button

4. **Activity Stats**
   - Total calls, SMS sent
   - Last contact date
   - Days since creation

### API Calls

- GET `/api/admin/leads/[id]` - Fetch lead with communications
- PATCH `/api/admin/leads/[id]` - Update lead status
- POST `/api/admin/leads/[id]/call` - Initiate call
- POST `/api/admin/leads/[id]/sms` - Send SMS

## Verification

- [ ] Page loads lead data correctly
- [ ] Communication timeline displays chronologically
- [ ] Status updates work
- [ ] Navigation from leads list works
- [ ] Responsive design
