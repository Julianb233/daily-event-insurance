# APPENDIX: DESIGN SPECIFICATIONS

## Color Palette

| Element | Color | Hex Code |
|---------|-------|----------|
| Primary Blue | Deep Blue | #1E40AF |
| Secondary | Teal | #0D9488 |
| Accent | Amber | #F59E0B |
| Success | Green | #10B981 |
| Background | Light Gray | #F9FAFB |
| Text | Dark Gray | #1F2937 |

## Typography

- **Headlines:** Bold, Sans-serif (Recommended: Inter, Poppins, or similar)
- **Body:** Regular weight, high readability
- **Emphasis:** Use color and bold, avoid italics for accessibility

## Iconography

Use simple, clean icons from:
- Heroicons
- Lucide Icons
- Font Awesome

## Layout Principles

1. Maximum 3 key points per slide
2. Large, readable text (minimum 24pt for body, 36pt for headlines)
3. Generous white space
4. Consistent alignment (left-aligned text preferred)
5. One clear takeaway per slide

## Animation Recommendations

- Keep animations minimal and purposeful
- Fade-in for new content
- No spinning, bouncing, or distracting effects
- Build slides progressively for complex information
