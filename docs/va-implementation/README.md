## Project Overview

**Daily Event Insurance** is a B2B embedded insurance platform enabling gyms, climbing facilities, and rental businesses to offer same-day coverage to their members.

**Tech Stack:**
- Website: Next.js 16, React 19, TypeScript, Tailwind CSS 4
- CRM: GoHighLevel (GHL)
- Integrations: Widget embed, REST API
