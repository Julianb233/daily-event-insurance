# Image Generation Prompts for Daily Event Insurance

Detailed prompts for Nana Banana Pro to generate all required images.

---

## Brand Guidelines for All Images

**Color Palette:**
- Primary Teal: #14B8A6 / #0D9488
- Dark Teal: #0F766E
- Background: White (#FFFFFF) or Light Gray (#F8FAFC)
- Accent Blue: #0EA5E9
- Text Dark: #1E293B

**Style Guidelines:**
- Clean, professional B2B aesthetic
- Modern, minimalist design
- High contrast for readability
- No excessive gradients or cosmic effects
- Human elements where appropriate (diverse, professional)

5. Test on website
