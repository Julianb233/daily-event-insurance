## Training Paths

### Path 1: Basic Support Agent (Days 1-7)

| Day | Focus | Documents | Goal |
|-----|-------|-----------|------|
| 1 | Onboarding | Getting Started, Platform Overview | System access, basic navigation |
| 2 | Communication | Chat Widget, Response Templates | First conversations |
| 3 | Knowledge | FAQ Management, Common Issues | Answer 80% of questions |
| 4-5 | Practice | All support-agent docs | Supervised live support |
| 6-7 | Independence | Review + certification | Solo support shifts |

**Certification:** Pass scenario-based assessment with 85% score

### Path 2: Technical Support Specialist (Weeks 2-4)

Requires completion of Path 1.

| Week | Focus | Documents | Goal |
|------|-------|-----------|------|
| 2 | Integrations | All integration guides | Debug POS issues |
| 3 | API Support | API Support Guide | Help developer partners |
| 4 | Advanced | All docs + PRDs | Complex troubleshooting |

**Certification:** Successfully resolve 10 technical escalations

### Path 3: Senior Support / Team Lead (Ongoing)

- QA and coaching training
- Escalation handling mastery
- Policy interpretation
- Team mentorship
