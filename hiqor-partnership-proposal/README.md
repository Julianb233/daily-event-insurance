# HIQOR Partnership Proposal

This folder contains all supporting documents for the Daily Event Insurance + HIQOR strategic partnership proposal.

## Documents

| File | Description |
|------|-------------|
| **email-template.html** | Email template to send to HIQOR stakeholders with partnership overview |
| **executive-summary.html** | One-page executive summary of the partnership (print-friendly) |
| **partnership-terms.html** | Detailed terms sheet with revenue share, pricing, and services |
| **platform-features.html** | Showcase of platform capabilities and features |

## Live Presentation

The full interactive presentation is available at:
**https://dailyeventinsurance.com/about/hiqor/presentation**

## Key Partnership Terms

- **Revenue Split**: 70% HIQOR / 30% Daily Event Insurance
- **Microsite Cost**: $600/month per site
- **Support**: Full customer support included
- **Onboarding**: Complete training program included
- **Sales Resources**: Marketing materials & tools included

## How to Use These Documents

1. **Email Template** - Copy the HTML and paste into your email client, or open in browser and copy content
2. **Executive Summary** - Print or share as PDF for quick overview
3. **Partnership Terms** - Share with legal/finance teams for review
4. **Platform Features** - Share with technical teams to understand capabilities
